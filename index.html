Zero Motorcycle Community

Overview

This organization supports owners of Zero electric motorcycles in an unofficial capacity, providing tools for working with the vehicles, mainly in data analysis.

Visit our Github organization page for details about our projects.

Tools

Logs

The HTML Zero Log File Web Parser and Viewer can be used online or offline, and provides visualizations directly.

The alternative Zero Log Parser is online-only but provides spreadsheet exports in Excel format or to Google Sheets.

OBD Gen2

OpenMBB is a free, open-source desktop app (Windows and Linux) that reads a Gen2 MBB Zero's live data straight off the OBD/service port — battery health, motor controller, error logs, settings, and ride history — then backs it up and analyzes it offline. It is read-first (any writes are gated behind login + confirm), includes a simulator so it can be tried with no bike, and has so far been verified on a 2017 FXS (MBB rev 41).

Gen3

Android App to keep using defunct Dashboard Navigation Overlays in Cypher III bikes: ZeroNavBridge. Supports Google Maps, Open Street Maps and Kurviger (experimental). For Developers: The BLE communication is seperated out into a library.
